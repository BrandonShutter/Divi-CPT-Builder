== Description ==

This plugin allows you to use Divi's Page Builder on all Custom Post Types.  This includes WooCommerce's Product pages, custom post types like Divi's Projects.

== Installation ==

1. Upload `divi_cpt_builder.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to any page, post, or cpt on your site and enjoy using the Divi Page Builder!
